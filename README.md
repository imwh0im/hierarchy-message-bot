# hierarchy-message-bot
---

Client: VueJS
Server: NestJS

## Description

[What's Hierarchy](https://en.wikipedia.org/wiki/Hierarchy) 

Use hierarchy structure for Low-level Message Bot.

---

## client

Frontend For Message Service.

### Installation
```bash
npm i
```

### Run (Delveopment)
```bash
npm run serve
```

### Deploy (Production)
```bash
npm run deploy
```

Backend For Message Service.

### Installation
```bash
$ npm i
```

### Run (Development)

### Deploy (Production)
```bash
npm i
```
